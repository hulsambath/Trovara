# /audit-deps

Dispatch the `deps-auditor` subagent to surface outdated and potentially vulnerable pubspec dependencies. Saves output to `docs/superpowers/audits/YYYY-MM-DD/05_perf_deps.md` (deps half only; perf half is `--full`-only via orchestrator).

## Steps

1. Resolve today's date and ensure the folder exists:
   ```bash
   mkdir -p "docs/superpowers/audits/$(date +%Y-%m-%d)"
   ```

2. Dispatch `deps-auditor` with prompt:

   > Audit pubspec dependencies per your standing instructions. Return only the markdown table.

3. Write the returned table to `docs/superpowers/audits/<today>/05_perf_deps.md`, prefaced with:

   ```markdown
   # Dependencies Audit — <today>

   Generated by `/audit-deps`. Perf hotspots are surfaced separately via `/audit --full`.

   <table from subagent>
   ```

4. Print counts by severity to stdout.

## Notes

- Strictly read-only — never runs `flutter pub upgrade` or modifies pubspec files. Open question §6.2 in the spec defaulted to read-only.
- Idempotent — overwrites `05_perf_deps.md` for the current day.
- If `/audit --full` also runs today, the perf half will be appended to this file by the orchestrator.
