# /audit-coverage

Dispatch the `test-gaps-auditor` subagent to map untested services / repositories / ViewModels, then save its output to `docs/superpowers/audits/YYYY-MM-DD/03_test_gaps.md`.

## Steps

1. Resolve today's date and ensure the folder exists:
   ```bash
   mkdir -p "docs/superpowers/audits/$(date +%Y-%m-%d)"
   ```

2. Dispatch `test-gaps-auditor` with prompt:

   > Map untested services, repositories, and ViewModels under `lib/` per your standing instructions. Return only the markdown table.

3. Write the returned table to `docs/superpowers/audits/<today>/03_test_gaps.md`, prefaced with:

   ```markdown
   # Test Coverage Gaps — <today>

   Generated by `/audit-coverage`. To write a missing test, invoke the `test-writer` subagent with the source file path.

   <table from subagent>
   ```

4. Print counts by severity to stdout.

## Notes

- Idempotent — overwrites `03_test_gaps.md` for the current day.
- Pair with the `test-writer` subagent to actually generate the missing tests.
