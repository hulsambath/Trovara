Buttons

All buttons use font-weight: 600, border-radius: full, and height 40px (default). Provide clear visual feedback on hover/active states.

All input fields use surface-container-low background with a subtle 1.5px border. Focus ring uses a 3px primary-tinted outline.

Tag Chips

Tags are the core taxonomy of Trovara. Each category has a fixed color palette. All chips use border-radius: full. Selected chips show the tag color on a tinted background.
