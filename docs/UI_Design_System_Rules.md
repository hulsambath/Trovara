# Design System

## Colors

Use AppColors only.
Never hardcode colors.

## Spacing

Use:

- Insets.xs = 4
- Insets.sm = 8
- Insets.md = 16
- Insets.lg = 24

## Typography

Use AppTextStyle only.

## Buttons

Use AppButton widget.
Do not use ElevatedButton directly.

## Responsive

Use ScreenUtil.
Avoid MediaQuery unless necessary.
